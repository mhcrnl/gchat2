 8 = Disable tree-view dotted connecting lines
16 = Show channel modes in titlebar when key is set
32 = Smaller treeview/userlist row spacing (compact mode) (2.8.6+)
To set multiple options at once, add the numbers up. For example, to enable options 2, 8, and 32, use: /set gui_tweaks 42
To toggle a specific option, use: /set -or gui_tweaks <n>
To enable a specific option, use: /set -on gui_tweaks <n>
